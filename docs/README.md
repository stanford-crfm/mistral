# Install Sphinx

If you don't already have Sphinx set up install it with `pip`.

```bash
pip install sphinx
```

The documentation has been built with version 4.0.2.

```bash
$ sphinx-build --version
sphinx-build 4.0.2
```

# Build The Docs

From `docs` directory run:

```bash
$ BUILDDIR=/path/to/build make html
```
